# AI-Powered Chatbot

## Description
This project is an AI-powered chatbot developed using Python and Flask. It responds to user queries based on predefined intents and can be used for customer support or FAQ systems.

## Technologies Used
- Python
- Flask
- Natural Language Processing (basic pattern matching)

## Features
- Accepts user messages
- Returns automated responses
- Simple and efficient chatbot logic

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Test using POST request:
   URL: http://127.0.0.1:5000/chat
   Body:
   {
     "message": "hello"
   }

## Outcome
A working AI chatbot that provides automated responses to user queries.
